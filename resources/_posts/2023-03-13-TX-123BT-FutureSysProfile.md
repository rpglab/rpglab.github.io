## Texas 123-bus Backbone Power System

This dataset includes future load profiles of the TX-123BT system.

The TX-123BT system represents a synthetic Texas 123-bus 345kV Backbone Transmission System (TX-123BT). For more details about this TX-123BT system, please check it here: https://rpglab.github.io/resources/TX-123BT/
